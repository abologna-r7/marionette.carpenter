Class: Controller

Defined in: src/controllers/application_controller.js.coffee
Inherits: Marionette.Controller

Overview

This is the base controller class that all other Marionette controllers should inherit from. It provides some convenience methods that most controllers would implement, like unbinding or adding views to the main region of the @app.
