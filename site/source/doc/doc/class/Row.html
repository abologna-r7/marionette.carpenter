Overview

Render a single row (per model in the collection)

Events:

table:row:selection_toggled

Emitted when this row's checkbox is toggled. Also emitted as selection_toggled on the model.

Parameters:

  • model ( Entities.model ) — the row's associated model

table:row:selected

Emitted when this row's checkbox is selected. Also emitted as selected on the model.

Parameters:

  • model ( Entities.model ) — the row's associated model

table:row:deselected

Emitted when this row's checkbox is deselected. Also emitted as deselected on the model.

Parameters:

  • model ( Entities.model ) — the row's associated model
