(Array<Object>) actionButtons
Array of option hashes for rendering the action buttons above the form. Should contain an array of Objects with the following properties:

  • label - [String] the text displayed in the button
  • click - [Function] Callback fn for the click event. Receives selectAllState, selectedIDs, deselectedIDs, 'selectedVisibleCollection, and 'tableCollection arguments
  • id - [String] a CSS id to apply to the button
  • class - [String] one or more CSS classes to apply to the button
  • containerClass - [String] one or more CSS classes to apply to the button's containing element
  • event - [String] an event name to trigger on the global event bus when the button is clicked
  • activateOn - [String] (any|one|many) enable/disable this button based on how many rows are selected. This is a noop if #selectable is false. If no option is passed, button will always remain enabled.
data used to render the table header buttons

(Array<Object>) columns
Array of option hashes for rendering the columns in the table. Should contain an array of Objects with the following properties:

  • label - [String] the text displayed in the column
  • attribute - [String] the model attribute that is displayed in this column
  • sortAttribute - [String] a separate string that is sent to the server as a sort key
  • sortable - [Boolean] enable the user to sort by this column (true)
  • defaultDirection - [String] (asc|desc) default sort direction
  • view - [Class] subclass of Backbone.View that will be rendered in this column
  • viewOpts - [Object] any options to pass to the view when it's instantiated
  • hoverView - [Class] subclass of Backbone.View that will be rendered as an overlay on hover
  • hoverOn - [Function] that returns Boolean of whether to display the HoverView on hover
  • render - [Function] callback function that accepts the model and returns HTML markup for rendering
